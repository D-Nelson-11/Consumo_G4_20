Vuelo

En el formulario de Vuelo podremos llevar un control específico de cada vuelo que transportará a los pasajeros, se detallará un codigo de vuelo, la ciudad de origen y destino, la fecha en que se realizará el vuelo, el tipo de avión y la cantidad de kilometros que se recorreran a lo largo del vuelo. También se podrá Agregar, Editar y Eliminar los registros que se encuentran en la base de datos, a continuación, una breve explicación de la función de cada uno de los campos

Avion

En este Fomulario podremos encontrar información de cada avión es decir; el tipo de avión las horas de vuelo, la capacidad de pasajeros con la que cuenta el avión, la fecha del primer vuelo, su país de construcción y la cantidad de vuelos que este ha realizado, además podremos Agregar, Editar y Eliminar los registros almacenados en la base de datos, y todo esto gracias a las siguientes funciones implementadas en backend:

  1. Cargar aviones: En este método se verá toda la información de los aviones.
  2. Cargar Avion: Esta Función buscara un avión en específico mediante su “NumeroAvion”.
  3. Agregar Avion: En este método se insertarán nuevos aviones con sus respectivos datos.
  4. Actualizar Avion: Este método es para actualizar o modifAvióncualquier información de los aviones.
  5. Eliminar Avion: Este método es para eliminar cualquier avión.

Reservación

El Formulario de Reservacion es un documento con un plan detallado con sus viajes a un país específico, en este formulario podremos agregar, editar y eliminar los diferentes datos de cada reservacion realizada, como ser:

  1. Id Reservacion: Es el ID único para cada Reservacion(no se puede editar)
  2. Codigo de Vuelo: Código específico para cada vuelo.
  3. Codigo de Pasajero: Código con el que se identificará cada Pasajero.
  4. Nombre de Pasajero: Datos personales del pasajero.
  5. Otros datos que se podrán encontrar, son la ciudad de destino, fecha de vuelo y sus respectivos precios

Pasajero

La Entidad Pasajero que se emplea en el formulario del mismo nombre(Pasajero), se crea con la finalidad de dar a conocer informacion especifica de las personas que solicitan viajar dicha informacion se divide en los siguientes campos:

  1. CodigoPasajero: Los ayuda a reconocer que pasajero es sin necesidad de saber el nombre.
  2. Nombres: Son los nombres de los pasajero.
  3. Apellidos: Son los apellidos del pasajero.
  4. Fecha Registro: Es la fecha en el cual el pasajero viajo por primera vez.
  5. Nacionalidad: El pais de nacimiento del pasajero.
  6. Numero de Telefono: Para tener una comunicacion directa con el pasajero.
  7. Email: Para una comunicacion anticipada se envia un email.
